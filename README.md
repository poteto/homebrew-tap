# Poteto Tap

## How do I install these formulae?

`brew install poteto/tap/<formula>`

Or `brew tap poteto/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "poteto/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
